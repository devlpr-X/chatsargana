# Чацаргана — Django веб сайт

УВС аймгийн чацарганы бүтээгдэхүүний веб сайт. Django + SQLite.

## Ажиллуулах

```bash
# 1. Хамаарлуудыг суулгах
pip install -r requirements.txt

# 2. Өгөгдлийн сан үүсгэх
python manage.py migrate

# 3. Анхны мэдээлэл оруулах
python manage.py seed_data

# 4. Admin хэрэглэгч үүсгэх
python manage.py createsuperuser

# 5. Сервер асаах
python manage.py runserver
```

Сервер ажилласны дараа:
- **Нүүр хуудас**: http://127.0.0.1:8000/
- **Админ самбар**: http://127.0.0.1:8000/admin/

## Анхны admin нэвтрэх

seed_data-аас өмнө үүсгэсэн бол:
- Нэвтрэх нэр: `admin`
- Нууц үг: `admin1234`

## Хуудсуудын URL

| Хуудас         | URL               |
|----------------|-------------------|
| Нүүр           | `/`               |
| Танилцуулга    | `/info/`          |
